# Nano Banana Logo Generation Prompt

## Primary Prompt (Use This First)

```
Create an elegant, modern European-style logo for "DRYEYELA" - a premium eye care e-commerce brand. 

Design requirements:
- Typography: Elegant serif or refined sans-serif font, European luxury brand aesthetic
- Style: Minimalist, sophisticated, clean - think Swiss design or Scandinavian minimalism
- Colors: Deep navy blue (#152c52) and bright blue (#009ade)
- Icon: Subtle, minimalist eye icon (optional, very refined if included)
- Layout: Horizontal logo with text "DRYEYELA" in elegant typography
- Mood: Premium, trustworthy, medical professionalism, modern European elegance
- Negative space: Generous spacing, breathing room
- Line weight: Thin, refined lines (0.5-1px)
- Opacity: Subtle, sophisticated use of transparency

The logo should feel like a European luxury brand - think Italian pharmaceutical elegance, Swiss precision, or Scandinavian minimalism. Clean, timeless, professional.
```

## How to Use with Nano Banana

1. **Go to**: https://www.ailogocreator.io/im/nanobanana or https://www.nanobananapro.fit/
2. **Paste** one of the prompts above
3. **Generate** multiple variations
4. **Download** the best result as SVG or PNG
5. **Replace** `/public/assets/logos/logo-dryeyela-new.svg` with the generated logo

## Tips for Better Results

- **Try multiple prompts**: Use different variations to see what works best
- **Iterate**: Refine based on results - add "more minimalist" or "thinner lines" if needed
- **Specify format**: Ask for "SVG vector logo" for scalability
- **Color codes**: Include hex codes (#152c52, #009ade) for exact brand colors
- **Negative prompts**: Add "no bold fonts, no heavy lines, no bright colors" if needed

## What to Look For

✅ **Good Results:**
- Clean, elegant typography
- Subtle, refined icon
- Generous spacing
- Professional appearance
- European luxury feel

❌ **Avoid:**
- Too bold or heavy
- Cluttered design
- Bright, garish colors
- Overly decorative
- American-style bold logos
